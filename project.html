Key functions

School mail authentication, document inquiry, document addition/edit request and discussion, document change history inquiry (Git-like UI), community (notification board, free bulletin board), reporting and warning system

Main content

It is a web service that provides useful information about the Seoul National University Central Club Sunshine Volunteer Corps in writing. I developed it based on the idea that I was sorry that there was no proper representative website compared to the size of the club at the time of the club activities.
It is a primitive form of web application that implements both the front and back ends in Django. This is a project that aims to fully understand the environment and technology currently adopted by the company.
